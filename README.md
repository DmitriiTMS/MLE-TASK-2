# Telegram бот для получения курса валют
Telegram бот для получения актуальных курсов обмена валют.

## Возможности
- Получение текущего курса для любой валютной пары
- Простой интерфейс команд
- Обработка ошибок при некорректном вводе и сбоях API
- Детальное логирование

## Требования
- Node.js (версия 14 или выше)
- npm
- Токен Telegram бота (получить у [@BotFather](https://t.me/botfather))

## Установка
1. Клонируйте репозиторий: git clone <repository-url>
2. Перейдите в папку с проектом
3. Установите зависимости: npm install
4. Создайте файл .env на основе .env.example
5. Добавьте токен вашего бота в файл .env:
6. Выполните команды:
    npm run build
    npm run start

## Тестирование:
    Выполните команду: npm run test
